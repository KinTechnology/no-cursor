# No-Cursor

A simple program that hides your cursor in X11.

## Getting Started

### Prerequisites

- A C compiler
- X11 development libraries (libxfixes-dev on Debian/Ubuntu)

### Building

```bash
make
```

## Usage

```bash
./nocursor
```
